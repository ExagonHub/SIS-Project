# SIS Project (Student Information System)

Version: v0.0.4  
Status: In Development  

## About
Terminal-based student information system built with Python.
This project is developed step-by-step to practice core programming fundamentals and system thinking.

## Current Features
- Simple login (demo-level, fixed credentials)
- Student lookup by ID
- Formatted student information display
- Grade average calculation
- Pass/Fail evaluation

## Data Rules
- Student ID must be unique (no overwrite on add)
- If grades are missing, averages should not be calculated (no crash)

## Planned (when learned)
- Menu system (loops)
- Add/update student (loops + validation)
- Data persistence (file I/O + JSON)
- Logging (file I/O)
